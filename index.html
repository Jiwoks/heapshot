Script options

You can init Heapshot with some options

  • indexStart is the z-index css property which is applied to the images, the count start from this option
  • rotation is the angle applied during easing
  • easing is an equation to calculate the easing (more functions here Easing equations
  • overflowparents will set all the parents overflow to visible if set to true
